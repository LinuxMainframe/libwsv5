\hypertarget{Disabling-example}{}\doxysection{Disabling}
Enable or disable a filter on a source.

Enable or disable a filter on a source.\+Filters are effects applied to sources -\/ color correction, blur, noise suppression, etc. Each filter can be enabled or disabled. Disabling a filter removes its effect without deleting it, so you can toggle it back on later.

Use this to\+:
\begin{DoxyItemize}
\item Dynamically control effects (blur camera when not looking at it)
\item Toggle noise suppression on/off for a microphone source
\item Enable/disable color correction for different lighting conditions
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
{\em conn} & Connection handle (must be in CONNECTED state) \\
\hline
{\em source\+\_\+name} & Name of the source containing the filter (case-\/sensitive) \\
\hline
{\em filter\+\_\+name} & Name of the filter to enable/disable (case-\/sensitive) \\
\hline
{\em enabled} & true to enable the filter, false to disable it \\
\hline
{\em response} & Optional pointer to receive response \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
OBSWS\+\_\+\+OK if request sent successfully 

OBSWS\+\_\+\+ERROR\+\_\+\+NOT\+\_\+\+CONNECTED if not connected 

OBSWS\+\_\+\+ERROR\+\_\+\+INVALID\+\_\+\+PARAM if source\+\_\+name or filter\+\_\+name is NULL
\end{DoxyReturn}
\begin{DoxyNote}{Note}
If the source or filter doesn\textquotesingle{}t exist, OBS returns an error. 

Changes are instant on stream/recording. 

You\textquotesingle{}ll get a Source\+Filter\+Enabled\+State\+Changed event when complete.
\end{DoxyNote}
