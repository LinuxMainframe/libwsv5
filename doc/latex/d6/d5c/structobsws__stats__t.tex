\doxysubsection{Detailed Description}
Connection statistics -\/ useful for monitoring and debugging connection health. 

These stats let you see what\textquotesingle{}s happening on the connection -\/ how many messages have been sent/received, error counts, latency, etc. Useful for monitoring the connection quality, detecting if something is wrong, or just being curious about protocol activity. You get these by calling \mbox{\hyperlink{library_8h_a5cce41cb6c70229eaab8b1b3afba57a9}{obsws\+\_\+get\+\_\+stats()}}.
