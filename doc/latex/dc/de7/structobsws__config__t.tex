\doxysubsection{Detailed Description}
Connection configuration structure. 

This structure holds all the settings for connecting to OBS. You should fill this out with your specific needs, then pass it to \mbox{\hyperlink{library_8h_a7ed7225961f0989acb31ebbd9c099626}{obsws\+\_\+connect()}}. A good starting point is to call \mbox{\hyperlink{library_8h_a1f1c69172bbf706ac225b718797d30c6}{obsws\+\_\+config\+\_\+init()}} which fills it with reasonable defaults, then only change the fields you care about (usually just host, port, and password).

Design note\+: We use a config struct instead of many function parameters because it\textquotesingle{}s more flexible -\/ adding new configuration options doesn\textquotesingle{}t break existing code. It also makes it clear what options are available.
